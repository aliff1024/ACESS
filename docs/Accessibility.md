\# Accessibility Preset Design Guide



These presets are starting templates only. Learners must always be able to customize individual settings after selecting a preset.



The system should automatically recommend a preset during onboarding, but never lock users into it.



\---



\# Age-Based Adaptation Rules



Accessibility needs remain relatively consistent across ages, but presentation style should adapt.



| Age Group   | Design Approach                                                                                                                                                     |

| ----------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------- |

| 6–12 Years  | Highly visual, icon-driven navigation, larger buttons, simplified layouts, gamification, rewards, badges, visual progress indicators, minimal text-heavy interfaces |

| 13–17 Years | Balanced approach between visual guidance and independence, moderate gamification, structured learning flows, clearer executive-function supports                   |

| 18+ Years   | Professional appearance, reduced gamification, productivity-focused tools, advanced customization, focus mode, dashboard-based learning experience                  |

\# Dyslexia Preset



\## Primary Goal



Reduce reading fatigue, visual crowding, and text-processing difficulty.



\### Reading Preferences



| Setting         | Recommended Value                     |

| --------------- | ------------------------------------- |

| Font Family     | Atkinson Hyperlegible, Arial, Verdana |

| Font Size       | 18px                                  |

| Line Spacing    | 1.6x                                  |

| Word Spacing    | +20%                                  |

| Letter Spacing  | +10%                                  |

| Text Alignment  | Left Aligned                          |

| Background Tint | Cream, Pale Yellow, Soft Blue         |



\### Interface Behaviour



| Component      | Behaviour                                        |

| -------------- | ------------------------------------------------ |

| Sidebar        | Larger text labels, fewer menu items per section |

| Lesson Content | Automatically split long paragraphs              |

| Quiz Questions | One question focus view available                |

| Activities     | Text-to-speech option visible                    |

| Resources      | Reading mode available                           |

| Dashboard      | Simplified card layout                           |



\### Age Adaptation



| Age   | Additional Features            |

| ----- | ------------------------------ |

| 6–12  | Picture-supported instructions |

| 13–17 | Reading support toolbar        |

| 18+   | Professional reading mode      |

\# ADHD Preset



\## Primary Goal



Reduce distractions and support attention, memory, and task completion.



\### Reading Preferences



| Setting         | Recommended Value |

| --------------- | ----------------- |

| Font Family     | Arial             |

| Font Size       | 18px              |

| Line Spacing    | 1.5x              |

| Background      | Soft Grey         |

| Chunked Content | Enabled           |



\### Focus Preferences



| Feature                       | Status |

| ----------------------------- | ------ |

| Reading Spotlight             | ON     |

| Focus Mode                    | ON     |

| Distraction-Free Mode         | ON     |

| Reduced Motion                | ON     |

| Persistent Progress Indicator | ON     |



\### Interface Behaviour



| Component     | Behaviour                         |

| ------------- | --------------------------------- |

| Sidebar       | Essential items only              |

| Dashboard     | Shows today's tasks first         |

| Lessons       | Multi-step learning flow          |

| Activities    | One activity at a time            |

| Quizzes       | Section-by-section                |

| Notifications | Consolidated instead of scattered |

| Assignments   | Broken into manageable steps      |



\### Executive Function Supports



| Feature             | Status |

| ------------------- | ------ |

| Task Checklist      | ON     |

| Progress Tracker    | ON     |

| Auto Save           | ON     |

| Resume Learning     | ON     |

| Visual Progress Bar | ON     |



\### Age Adaptation



| Age   | Additional Features                        |

| ----- | ------------------------------------------ |

| 6–12  | Rewards, badges, achievement animations    |

| 13–17 | Goal tracking and study streaks            |

| 18+   | Productivity dashboard and focus analytics |

\# Autism Preset



\## Primary Goal



Increase predictability, structure, and reduce uncertainty.



\### Reading Preferences



| Setting        | Recommended Value |

| -------------- | ----------------- |

| Font Family    | Arial             |

| Font Size      | 18px              |

| Background     | Soft Blue         |

| Reduced Motion | ON                |



\### Sensory Preferences



| Feature            | Status    |

| ------------------ | --------- |

| Dark Mode Option   | Available |

| Muted Colors       | ON        |

| Animations         | None      |

| Consistent Layouts | Required  |



\### Interface Behaviour



| Component     | Behaviour                          |

| ------------- | ---------------------------------- |

| Sidebar       | Fixed position, never rearranges   |

| Dashboard     | Consistent layout across courses   |

| Lessons       | Predictable structure              |

| Activities    | Clear expectations before starting |

| Quizzes       | Explicit instructions              |

| Navigation    | Consistent labels everywhere       |

| Notifications | Calm and non-intrusive             |



\### Executive Function Supports



| Feature                   | Status |

| ------------------------- | ------ |

| Visual Schedule           | ON     |

| Progress Timeline         | ON     |

| Section Transition Alerts | ON     |

| Task Preview              | ON     |



\### Example



Before entering a new section:



"Section A completed."



"Next: Section B (Estimated time: 10 minutes)"



\### Age Adaptation



| Age   | Additional Features                   |

| ----- | ------------------------------------- |

| 6–12  | Visual schedules with icons           |

| 13–17 | Structured daily learning plans       |

| 18+   | Detailed course roadmap and timelines |

\# Universal Features For All Presets



These should be available regardless of selected preset.



| Feature                          | Requirement       |

| -------------------------------- | ----------------- |

| Font Customization               | User can override |

| Theme Customization              | User can override |

| Accessibility Preset Switching   | Any time          |

| Dark Mode                        | Available         |

| Light Mode                       | Available         |

| Keyboard Navigation              | Supported         |

| Screen Reader Compatibility      | Supported         |

| Auto Save                        | Available         |

| Mobile Accessibility             | Required          |

| Cognitive Accessibility Settings | Editable anytime  |



The selected preset should influence not only styling but also layout structure, navigation complexity, lesson rendering, activity behavior, and executive-function support throughout the entire LMS.
